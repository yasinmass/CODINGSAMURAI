# 🚀 Coding Samurai Internship – Data Science Projects

## 📌 Overview
This repository contains all the projects completed as part of the **Coding Samurai Data Science Internship Program**.

The goal of this internship is to gain practical experience in data analysis, visualization, and machine learning by working on real-world datasets.

## 🎯 Internship Details

---

## 📂 Projects Included

### 📊 1. Titanic Exploratory Data Analysis (EDA)
- Performed data cleaning and preprocessing
- Analyzed survival patterns
- Created visualizations (countplots, histograms, heatmaps)

🔍 Key Insights:
- Only ~38% passengers survived
- Majority of passengers were in 3rd class
- Females had higher survival rate
- Fare distribution is highly skewed

---

### 🤖 2. Titanic Survival Prediction (Logistic Regression)
- Built a classification model to predict survival
- Performed feature selection and encoding
- Evaluated model using accuracy and confusion matrix

📈 Results:
- Achieved ~75%–85% accuracy
- Identified key factors affecting survival
