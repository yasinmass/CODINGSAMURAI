# Titanic Dataset - Exploratory Data Analysis (EDA)

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to understand patterns, relationships, and key factors affecting passenger survival.

## 📊 Steps Performed
- Data loading and inspection
- Handling missing values
- Data visualization (countplots, histograms)
- Correlation analysis using heatmap

## 🔍 Key Insights
- Only ~38% passengers survived
- Majority of passengers were in 3rd class
- Females had higher survival rate than males
- Fare distribution is highly skewed
- Most passengers were between age 20–38

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📁 Dataset
Titanic dataset loaded from Seaborn library

## 🚀 Conclusion
EDA helps in understanding data patterns and preparing it for machine learning models.
